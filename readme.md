# mongo-into

The most basic CRUD you can do, using mongodb and mongoose.

## to run

1. install packages: `npm install`

2. (optional) run seed file: `./seed_run.sh`

2. start mongo: `mongod --dbpath ~/data/db` (I don't like running it from my root dir.)

3. start server: `nodemon`

4. open browser to localhost:3000
